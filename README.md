\# Spring Week Robotics Internship – VEX Robotics



This repository documents my spring week robotics internship, where I built and tested VEX IQ robotics projects.



The internship was hardware-focused, so we did not write code. Instead, I gained hands-on experience with mechanical assembly, robot design, testing, teamwork, and problem solving.



\## Project Lineup



During the internship, we built several VEX robotics projects.



!\[Lineup of VEX robotics projects](media/vex-project-lineup.jpg)



\## VEX Slick Robot



One of the robots I worked on was based on the VEX IQ \*\*Slick\*\* build.



Slick is a pusher-style robot with a front plow extension that can collect and push objects.

\## Skills Practiced



\- Mechanical assembly

\- Building with VEX IQ robotics parts

\- Robot prototyping

\- Testing and improving robot designs

\## Reflection



This internship helped me understand robotics from a practical engineering perspective. Even though we did not code, I learned how physical design choices affect robot movement, stability, and performance.
